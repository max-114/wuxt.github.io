eslint报错：vue/multi-word-component-names

2022-04-07 解决方案 100 words
eslint

修改 .eslintrc.js文件配置

配置后需要重新编译

忽略指定文件名

module.exports = {
  ...
  rules: {
    'vue/multi-word-component-names': [
      'error',
      {
        ignores: ['index'] // 配置需要忽略的文件名
      }
    ]
  }
}

忽略所有文件名

module.exports = {
  ...
  rules: {
    'vue/multi-word-component-names': ['off']
  }
}

检测文件名，通过警告提醒，不影响编译

module.exports = {
  ...
  rules: {
    'vue/multi-word-component-names': ['warn']
  }
}
