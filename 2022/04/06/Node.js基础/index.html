Node.js 基础

第一章 npm

1.1 node 包版本

1.1.1 包的版本号

13.4.6

  • major（主版本）: 13
  • minor（次版本）:4
  • patch:6

1.1.2 包的版本符号

^13.0.0

  • ^表示锁定主版本号为 13，其余版本更新时采用最新版

~13.4.0

  • ~表示锁定主版本号和次版本号
  • 使用最新版本

1.2 常用命令

安装包

npm i
// 全局安装
npm install -g
// 安装开发依赖
npm i xxx -D
// 安装生产依赖
npm i xxx -S
// 安装指定版本包
npm i package_name@version

查看包的所有版本

npm view package_name versions

查看过期的包

npm outdated

升级包

npm update

清除缓存

npm cache clean --force

查看当前项目引用的包

npm ls

1.3 npm 脚本

1.3.1 执行顺序

"scripts": {
  "script1": "node script1.js",
  "script2": "node script2.js"
}

如果是并行执行（执行顺序不确定），可以使用 & 符号

npm run script1 & npm run script2

如果是继发执行（即只有前一个任务成功，才执行下一个任务），可以使用 && 符号

$ npm run script1 && npm run script2

1.3.2 简写形式

npm start => npm run start

1.3.3 变量

npm 脚本有一个非常强大的功能，就是可以使用 npm 的内部变量

通过 npm_package_ 前缀，npm 脚本可以拿到 package.json 里面的字段

注意：一定要在 npm 脚本中运行（如：npm run view）才可以，直接在命令行中运行 JS（如：node view.js）是拿不到值的

{
  "name": "foo",
  "version": "1.2.5",
  "scripts": {
    "view": "node view.js"
  }
}
// view.js
console.log(process.env.npm_package_name) // foo
console.log(process.env.npm_package_version) // 1.2.5
